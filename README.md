# Mamba

* Author: Coleman Word 
* Email: coleman.word@gofunct.com
* Download: `go get github.com/gofunct/mamba
* Description: gRPC service to execute shell commands transparently on a remote server

## Usage
```text
Mamba v0.1.1

ğŸ A snake powered boilerplate generator

Usage:
  mamba [command]

Available Commands:
  app         ğŸ Generate boilerplate to bootstrap an application
  contain     ğŸ A Docker development utility
  ctl         ğŸ A Kubernetes development utility
  edit        ğŸ A edit templates for protoc output
  help        Help about any command
  load        ğŸ Download single, multiple, or all files from a remote repository
  script      ğŸ One line golang scripts
  walk        ğŸ Walk a filepath with a given function and file extension

Flags:
  -h, --help   help for mamba

Use "mamba [command] --help" for more information about a command.
```
